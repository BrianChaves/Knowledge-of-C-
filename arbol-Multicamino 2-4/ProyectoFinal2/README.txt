Este es un proyecto concluido de de un arbol multi caminos 2-4,
en este se muestra un menu en  ingles y en las clases se van a encontrar comentarios en ingles:

-----------------Menu de Arbol Multi-Camino 2-4----------------------
( 1 )   Insert n random Integers
( 2 )   Perform 2n operations with probability
( 3 )   Print In Order
( 4 )   View Tree level by level
( 5 )   Insert an element
( 6 )   Delete an element
( 7 )   Find an element
( 8 )   User Manual
( 0 )   Get out

Se va a encontrar un manual de los elementos anteriores y
una explicacion mas detallada de cada uno de ellos del menu.
